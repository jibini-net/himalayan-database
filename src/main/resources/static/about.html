About the Visualization of the Himalayan Database
This is a collection of tools and web panels for accessing the Himalayan Database. It provides a way to connect to the online archive and visualize peak/expedition data. This project was started by the Eagle Apprenticeship program at UW-La Crosse and is supervised by Dr. Samantha Foley and written by Heather Miller and Zach Goethel.
The interactive peak map was created using Leaflet.js. Peak coordinates were retrieved from the Alpine Club Library and are subject to possible inconsistencies with the peak data from the Himalayan Database.
Graphs on the Explore Peaks page were created using Chart.js.
The Visualization of the Himalayan Database is maintained using Spring Boot, Apache Tika, and the JSON library.
